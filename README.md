# ⚽ Football Stadium

A modern and interactive **3D Football Stadium** built with **React, Vite, JavaScript, and Three.js**. Experience a realistic virtual football stadium with immersive lighting, detailed environments, smooth camera controls, and high-performance rendering directly in your browser.

---

## 🌟 Features

- ⚽ Interactive 3D Football Stadium
- 🎮 Smooth Camera Navigation
- 🌍 Realistic Stadium Environment
- 💡 Dynamic Lighting & Shadows
- 🏟️ High-Quality Stadium Model
- 🎨 Modern Responsive UI
- 🚀 Fast Performance with Vite
- 📱 Responsive Design
- 🎥 Immersive Viewing Experience
- 🔥 Optimized Rendering
- 🌙 Beautiful Visual Effects

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/rsamwilson2323-cloud/football-stadium-main.git
```

Go to the project folder

```bash
cd football-stadium-main
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
